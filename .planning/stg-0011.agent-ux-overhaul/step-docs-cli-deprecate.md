# stg-0011 — docs-cli-deprecate

## Цель

Зафиксировать в `docs/`, что CLI-интерфейс помечен deprecated: MCP — основной канал, CLI остаётся под REPL и старые команды, новые фичи в CLI не добавляются, последующий выпил CLI запланирован.

## Файлы

`docs/` (через MCP):
- Section id=35 (`CLI-интерфейс`) — добавить узел type=statement и/или type=note с deprecation-маркером.
- Возможно — обновить `text` существующих узлов в section, чтобы избежать впечатления, что CLI продолжает развиваться.

## Действия

1. `create-node` type=statement path=35 title=`CLI deprecated` text=описание: CLI больше не развивается, MCP — основной канал, REPL остаётся как dev-utility, удаление CLI запланировано.
2. Опционально — `create-node` type=note path=35 title=`Roadmap выпила CLI` text=что и в какой последовательности будет удалено.
3. Sweep по другим разделам `docs/`, упоминающим CLI как основной канал — обновить через `update-node` (упоминать MCP первым).

## Риски

Нет — текст в `docs/` отражает решение, без воздействия на код.
