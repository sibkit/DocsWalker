## Действия
1. Одним вызовом `git checkout ce3f05e -- <4 файла>` восстановить содержимое из коммита `ce3f05e` (последнего перед первым axes-коммитом `4baa335`).
2. Проверить через `git status` и `git diff HEAD --stat`, что в working tree откачены ровно эти 4 файла, без побочных изменений.
3. Бегло проверить grep'ом, что в откаченных файлах не осталось axis-терминологии (`axis`, `axis_definition`, `axes:`, `default_axis`, `system_axis`).
